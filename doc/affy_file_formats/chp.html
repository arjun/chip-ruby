Affymetrix® CHP Data File Format


  CHP FILE  

 

   Description
The CHP file contains probe set analysis results generated from Affymetrix software. There are several versions of CHP files generated by the MAS5, GCOS and other Affymetrix software.
  • CHP files generated by MAS5 and GCOS 1.0 and 1.1 software. Known as the "historical" CHP files.
  • CHP files generated by GCOS 1.2, 1.3 and 1.4 and BRLMM Analysis Tool. Also known as the XDA CHP files.
  • CHP files generated by the Expression Console software.
  • CHP files generated by the Genotyping Console software.
